## Test Task

### Prerequisite

> MongoDB

> NodeJS

> npm

### Clone Repo

```
git clone https://github.com/anandchapter247/user-product-view-demo.git
```

### head to local repo

```
cd user-product-view-demo/
```

#### Install dependencies

```
npm install
```

### Setup Fake Data

```
node dump.js
```

### Start the server

#### With Nodemon-Cli

```
npm run dev
```

#### With Node-Cli

```
node app.js
```

<p>See code in action. Go to <a href="http://localhost:8000">localhost:8000</a></p>
